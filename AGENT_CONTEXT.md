# Agent Context -- Dolev AI Seed Proposal

Use this file to bootstrap a new AI agent session on this proposal.

## Project overview

This is a **12-month seed proposal** (100k NIS) for the Hebrew University's AI
research programme. It proposes using a CNN trained on DXA-labelled dislocations
in grain interiors to identify "dynamic interfacial motifs" inside grain
boundaries where DXA fails.

- **PI:** Dr. Menahem Krief (Racah Institute of Physics, HUJI)
- **Co-I:** Prof. Yinon Ashkenazy
- **Student:** Dolev Ronen (M.Sc. in Physics)
- **Goal:** Feasibility study; successful outcome leads to a PhD proposal.

## Key files and what to know about them

### main.tex

- 162 lines. Sections: Background/question/innovation, Research hypothesis,
  Proposed use of AI, Data types and scope, Timeline, Applicants, Budget,
  Future plans.
- Body text fills ~2.8 pages; references (2-column footnotesize) fill the rest
  of page 3 and page 4. **Total must not exceed 4 pages.**
- No `\subsection` commands -- inline bold headings only.
- No em-dashes (`---`) or en-dashes (`--`) anywhere; replaced by commas,
  parentheses, or slashes.
- Abbreviations defined at first use: DXA, GB, CMG, CNN, ILDA, DDD.
  Standard abbreviations left unexpanded: FCC, BCC, MD, EAM, EBSD, TEM, ISF,
  PI, Co-I, ML, GPU, CPU.

## Constraints and style rules

1. Total document length: **4 pages max** (body + references).
2. Body text target: **slightly under 3 pages**.
3. Language: direct, modest, no AI-sounding prose. No dashes.
4. Citations: abbreviated journals, max 3 authors + et al.
5. Compute: MD on CPU cluster (up to 2048 cores), CNN training on GPU.
   Both provided by ICPL -- no compute budget requested.
6. Budget: 84k NIS scholarship + 16k NIS travel = 100k NIS total.
